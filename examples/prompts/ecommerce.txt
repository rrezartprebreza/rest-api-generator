Create an API for Category with:
- name (string, required, unique)
- description (string)

Create an API for Product with:
- name (string, required)
- description (string)
- price (decimal, required, min 0)
- stock (integer, required, min 0)
- status (enum: DRAFT, ACTIVE, ARCHIVED)
- belongs to Category
- has many Tag (many-to-many)

Create an API for Tag with:
- name (string, required, unique)
- description (string)

Create an API for Order with:
- orderNumber (string, unique)
- totalAmount (decimal, min 0)
- status (enum: PENDING, CONFIRMED, SHIPPED, DELIVERED)
- belongs to Customer
- has many OrderItem

Create an API for OrderItem with:
- quantity (integer, min 1)
- unitPrice (decimal, min 0)
- subtotal (decimal)
- belongs to Order
- belongs to Product

Create an API for Customer with:
- email (string, required, unique, valid email)
- firstName (string, required, min 2, max 50)
- lastName (string, required, min 2, max 50)
- has many Order
